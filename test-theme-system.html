How to Use

In Your App:

  1. Theme Context: The theme system is available through the useTheme() hook
  2. Settings Page: Users can change themes in the Settings page under "Appearance"
  3. Auto-Save: Theme preferences are automatically saved to localStorage
  4. System Default: Auto mode follows the user's system preference
  5. CSS Variables: All colors are available as CSS custom properties

Available Themes:

  • Professional: Clean black and white (default)
  • Ocean Blue: Professional blue theme
  • Forest Green: Natural green theme
  • Sunset Orange: Warm orange theme
  • Royal Purple: Elegant purple theme
  • Midnight Dark: Dark professional theme
